Clustering Coefficient

Hermetyczność środowiska — klika vs broker

Lokalny Współczynnik Klasteryzacji:
CC(v) = 2T(v) / (k(v) · (k(v) − 1))

T(v) = liczba trójkątów zamkniętych przy węźle v
k(v) = liczba sąsiadów (degree nieważony)
  • CC(v) ∈ [0, 1]. CC = 1 → każdy sąsiad zna każdego innego sąsiada = pełna klika
  • CC = 0 → żaden z sąsiadów nie jest ze sobą połączony = gwiazda
  • Wysoki CC = środowisko hermetyczne. Niski CC przy wysokim degree = broker łączący odrębne klastry
